Shuffled Text Generator

The Shuffled Text Generator randomly shuffles words in a selected text excluding punctuation marks.

The dialog box of the Shuffled Text Generator contains Basic Settings and custom settings. You can customize the generator by changing the basic and custom settings.

The dialog box of the Shuffled Text Generator also contains a text editor window, in which you can either manually type the text or load it from a text file.

Custom Settings

Min length

Set minimum string length of a field.

Max length

Set maximum string length of a field.

Load text from a file

Click to import text from a text file (.txt). All files filter (.*) is available.
