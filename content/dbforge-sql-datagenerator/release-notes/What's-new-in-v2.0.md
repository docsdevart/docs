# What's New in dbForge Data Generator for SQL Server v2.0

**Basic generators**

* Regular Expression  generates values based on a regular expression
* Weighted List  generates values from a simple (or regexp, or nested, or ranged) values list with specified frequency of occurrence
* SQL Query  generates values from a custom query
* Table or View  generates values from a table or view
* Text File  generates values from specified text file
* Files Folder  generates values by retrieving documents content from specified folder
* Shuffled Text  generates dummy text by shuffling words in specified text
* Lorem Ipsum  generates dummy text with some characteristics of real written text

**Other Features and Improvements**

* Support for command-line interface
* Significantly increased data population performance
